Basics of Information Retrieval

### How to run?

Можно создать виртуальное окружение или использовать глобальное*

Для запуска нужно установить две библиотеки - команда для их установки:

```
pip install requests beautifulsoup4 
```

Для глобальных окружений к команде выше могут понадобиться префиксы

```python3 -m <pip command>``` или ```py -m <pip command>```

Затем можно запустить приложение из директории ```src/```

```
py .\main.py python
```

или

```
python3 main.py py
```
